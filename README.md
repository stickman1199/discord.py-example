# [discord.py](http://discordpy.rtfd.io/en/latest/)

### Warning
Python 3.8 or higher is required.

### Installation
```python
# Linux/macOS
python3 -m pip install -U discord.py

# Windows
py -3 -m pip install -U discord.py
```
